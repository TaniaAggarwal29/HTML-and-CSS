Google Fonts

There are many fonts that are available in windows or in mac but not in other OS. In which case our selected font family might not work.
In that case we can import fonts from Google.

Go to Google Fonts

Select the category of fonts -> Select the font style you like -> Select the styles available for that particular font

This will open a 'selected family' tab on the side.

From there, either select the <link> option, copy the link given and paste it inside the head section of html file OR select the import option, copy the given content inside of style tag and paste it on the top of css file.

copy the style from "CSS rules to specify family" from same place and paste it inside the css file's body tag.
